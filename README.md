# Simple pixi sketch board App

demo for pixi debugging or performance profiling. easy to build, easy to run.

usage:

```
yarn
yarn dev
yarn build
```

附带一个调试UI，进行相关功能的配置，常用按钮说明：

- `Brush Type`用于调整画笔或切换橡皮擦
- `Record Data` 按钮点击后，会将当前画布的绘制信息存储在localstorage中
- `Draw Once` 或 `Draw Loop`，测试单次绘制及循环擦除绘制
- `Clear` 清空画布

![示意图](2021-07-08-19-22-03.png)

Native API，可用于与载体环境进行通信:

```
// 传递绘制数据，可用于从na获取路径数据，入参为数组
window.PadAPISendData([])
// 清空画板，可用于从na清空画板
window.PadAPIClear()
```
